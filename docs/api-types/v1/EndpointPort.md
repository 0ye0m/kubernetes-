###EndpointPort###

---
* name: 
  * **_type_**: string
  * **_description_**: name of this port
* port: 
  * **_type_**: integer
  * **_description_**: port number of the endpoint
* protocol: 
  * **_type_**: string
  * **_description_**: protocol for this port; must be UDP or TCP; TCP if unspecified
