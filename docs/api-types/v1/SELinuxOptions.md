###SELinuxOptions###

---

  * **_type_**: string

* role: 
  * **_type_**: string
  * **_description_**: the role label to apply to the container; see http://releases.k8s.io/HEAD/docs/labels.md

  * **_type_**: string
  * **_description_**: the type label to apply to the container; see http://releases.k8s.io/HEAD/docs/labels.md

  * **_type_**: string
